# Model limits

This tutorial demonstrates geometry propagation and measurement code.

It does not convert model controls into equipment settings. Gas flow, chamber
pressure, source power, bias, plating current, additive concentration, slurry,
and polish pressure require separate calibration.

The compact copper step uses directional geometric growth. It does not model
electrochemistry, current distribution, or additive kinetics. The final plane
cut does not model pad contact, pressure, or slurry selectivity.

The example is a two-dimensional trench-like section. A cylindrical
three-dimensional via can change particle transport and void topology.
